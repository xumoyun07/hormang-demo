# ✅ 2. Marketplace Health (TOP KPIs)

Use real request + offer data.

## Calculate:

* Active requests (status: open)
* Requests with 0 offers
* Avg offers per request
* Time to first offer

---

## Example logic:

```ts id="market_logic"
const activeRequests = requests.filter(r => r.status === "open");

const requestsWithoutOffers = activeRequests.filter(
  r => !offers.some(o => o.requestId === r.id)
);

const avgOffers =
  totalOffers / totalRequests;
```

---

## Display:

```text id="market_ui"
Active Requests | No Offers ⚠️ | Offers/Request | First Offer Time
```

# ✅ 4. Monetization Snapshot

Use `tangaTransactions`.

## Calculate:

* revenue (sum of purchases)
* Tanga sold (purchase type)
* Tanga spent (spend type)
* avg spend per provider

---

```ts id="money_logic"
const revenue = transactions
  .filter(t => t.type === "purchase")
  .reduce((sum, t) => sum + t.amount, 0);
```
