You are working on a React + TypeScript + Tailwind + Supabase (or localStorage fallback) service marketplace project in Replit.

## 🎯 Goal

Implement a full **Announcements System (News & Events)** that allows admins to create, manage, and broadcast content to users, and display it in `/provider-home`.

This system acts as a **mini CMS for communication, growth, and monetization messaging**.

---

# 🧠 Core Concept

Announcements = structured content:

* News → informational (updates, features)
* Events → action-driven (promos, campaigns)

---

# ✅ 1. Data Model

Create:

```ts
type Announcement = {
  id: string;
  type: "news" | "event";

  title: string;
  content: string;
  image?: string;

  ctaText?: string;
  ctaLink?: string;

  target: "all" | "providers" | "customers";

  isPinned?: boolean;
  expiresAt?: string;

  status: "draft" | "published";

  createdAt: string;
};
```

---

## Storage

## Fetch announcements:

Filter:

```ts
announcements.filter(a =>
  a.status === "published" &&
  (a.target === "all" || a.target === "providers") &&
  (!a.expiresAt || new Date(a.expiresAt) > new Date())
);
```
