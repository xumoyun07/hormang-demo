# ✅ 1. Popularity Ranking Logic

Calculate popularity from real marketplace activity.

Each category should get a popularity score based on actual platform data.

Use weighted ranking from:

### Primary:

* number of customer requests created in that category

### Secondary:

* number of offers submitted in that category

### Optional boost:

* number of completed orders in that category

---

# Suggested formula

```ts
popularityScore =
(requestCount * 1.0)
+ (offerCount * 0.5)
+ (completedCount * 0.8)
```

Replit can tune weights if needed.

* requests
* offers
* accepted/completed jobs
