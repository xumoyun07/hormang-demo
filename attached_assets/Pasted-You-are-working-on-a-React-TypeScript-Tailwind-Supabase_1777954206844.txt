You are working on a React + TypeScript + Tailwind + Supabase (or localStorage fallback) service marketplace project in Replit.

## 🎯 Goal

Build a **clean, modern, fully functional Customer Dashboard** at:

```text
/dashboard
```

This will replace the current landing page for **logged-in customers** and must be fully connected to real platform data (NO mock data).

* Mobile-first design
* Blue theme (primary: `#3B82F6`)

* Action-oriented UX (help user take next step quickly)

# 🔐 3. Role-Based Routing (CRITICAL)

After login:

```ts
if (user.role === "customer") → redirect to /dashboard

```

# 🧩 4. Data Connections (MANDATORY)

ALL sections must use REAL data:

## Requests:

```ts
requests.filter(r => r.userId === currentUser.id)
```

## Active Requests:

```ts
status !== "completed"
```

## Offers:

```ts
offers.filter(o =>
  userRequests.some(r => r.id === o.requestId)
)
```

## Chats:

* conversations involving current user
* sorted by last message
