You are working on a React + TypeScript + Tailwind + Supabase (or localStorage fallback) service marketplace project.

## 🎯 Goal

Implement a full **“Takliflar va Shikoyatlar” (Feedback System)** that allows both **customers and providers** to submit:

* 🆘 Muammo (Problem / Bug)
* ⚠️ Shikoyat (Complaint)
* 💡 Taklif (Suggestion)

This system must include:

* User submission UI
* Database structure
* User history view
* Full Admin Panel integration

---

# 🧠 1. DATABASE DESIGN

Create a new table:

```ts id="zbnfjk"
feedbacks
```

---

## Fields:

```ts id="syo17o"
id: string
userId: string
userRole: "customer" | "provider"

type: "problem" | "complaint" | "suggestion"

title: string
description: string

targetType?: "provider" | "customer" | "platform"
targetId?: string

relatedRequestId?: string
relatedOfferId?: string

attachments?: string[] // URLs (images/videos/files)

status: "new" | "in_review" | "resolved" | "rejected"
priority: "low" | "medium" | "high"

adminNote?: string

createdAt: string
updatedAt: string
```

---

## Storage:

* Use Supabase if available
* Otherwise fallback to localStorage
