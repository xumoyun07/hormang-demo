# ✅ 2. Data Model (Unify Transactions)

Create unified transaction model:

```ts id="transaction_type"
type TangaTransaction = {
  id: string;
  userId: string;
  amount: number; // + or -
  type: "purchase" | "spend" | "referral_reward" | "admin_adjustment";
  source?: string; // plan name, requestId, referral user, etc.
  balanceAfter: number;
  createdAt: string;
};
```

---

## Storage

* Supabase table OR localStorage:

```ts id="transaction_storage"
tangaTransactions: TangaTransaction[]
```
