You are working on a React + TypeScript + Tailwind + Supabase (or localStorage fallback) marketplace platform called Hormang.

## 🎯 Goal

Implement a **Customer Request Cooldown System** to prevent spam requests, fake demand generation, and marketplace abuse.

Customers should NOT be able to create unlimited requests rapidly.

The system must:

* protect providers from spam
* maintain request quality
* still allow normal users to use platform comfortably
* integrate cleanly with existing request creation flow

---

# 🧠 Core Marketplace Rule

After creating a request, customer must wait before creating another one.

---

# ✅ Recommended Cooldown Rules

## Default cooldown:

```text id="jlwm13"
5 minutes
```

between request creations.

---

# Additional protection:

If customer creates many requests quickly:

```text id="q8jlwm"
3 requests within 24h
```

→ activate longer cooldown:

```text id="jlwm14"
30 minutes cooldown
```

---

# 🚀 Why This Works

This prevents:

* fake request spam
* providers wasting Tanga
* artificial marketplace inflation
* testing abuse
* low-quality users

WITHOUT harming normal users.

---

# ✅ 1. Backend Logic (CRITICAL)

Before creating request:

Find customer's latest request.

Example:

```ts id="jlwm15"
const latestRequest = requests
  .filter(r => r.customerId === user.id)
  .sort((a, b) =>
    new Date(b.createdAt).getTime() -
    new Date(a.createdAt).getTime()
  )[0];
```

---

# Validation

```ts id="jlwm16"
const cooldownMs = 5 * 60 * 1000;

if (
  latestRequest &&
  Date.now() - new Date(latestRequest.createdAt).getTime() < cooldownMs
) {
  throw new Error("So'rovlar oralig'ida kutish vaqti mavjud");
}
```

---

# ✅ 2. Advanced Anti-Spam Layer

Count requests in last 24h:

```ts id="jlwm17"
const recentRequests = requests.filter(
  r =>
    r.customerId === user.id &&
    Date.now() - new Date(r.createdAt).getTime() < 24 * 60 * 60 * 1000
);
```

---

# If count >= 3

Activate extended cooldown:

const cooldownMs =
