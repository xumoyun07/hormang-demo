### 1. Data Structure (use localStorage)
- "customerRequests" → array of requests created by the logged-in customer
  Each request contains: id, category, answers (including urgency + budget), status, createdAt, city
- "offers" → array of offers received
  Each offer: { id, requestId, masterId, masterName, price (so'm), message, createdAt, averageResponseTime }
- "chats" → simple chat storage
  Each chat: { id, requestId, masterId, masterName, messages: [{sender: "customer"|"master", text, timestamp}] }

- `/chat/:chatId` → Chat page with specific master (or create dynamic chat)
