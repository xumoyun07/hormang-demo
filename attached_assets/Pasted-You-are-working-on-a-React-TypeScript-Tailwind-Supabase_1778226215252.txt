You are working on a React + TypeScript + Tailwind + Supabase (or localStorage fallback) marketplace platform called Hormang.

## 🎯 Goal

Implement a protected and scalable **Offer Limiting System** that prevents customer abuse and protects providers from wasting Tanga.

The platform rule is:

```text id="jlwm7h"
One customer request can accept ONLY ONE provider.
```

The new system must include:

* active offer limit
* lifetime offer limit
* auto-closing after acceptance
* provider-side visibility
* anti-abuse protections
* proper Tanga validation order

---

# 🧠 Core Marketplace Rules

## Rule 1 — Active Offer Limit

Each request can have maximum:

```text id="jlwm8m"
5 ACTIVE offers
```

Active statuses:

* pending
* negotiating
* accepted

---

# Rule 2 — Lifetime Offer Limit

Each request can receive maximum:

```text id="l4okmc"
10 TOTAL offers EVER
```

INCLUDING:

* rejected
* cancelled
* expired

This prevents customers from endlessly cycling offers.

---

# Rule 3 — One Accepted Offer Only

When customer accepts one provider:

```text id="g4wteq"
request.status = "matched"
```

OR:

```text id="z1d8oz"
request.acceptedOfferId = offer.id
```

Then:

* request permanently closes for new offers

* all other pending offers receive:

  * Mijoz boshqa ijrochi taklifini qabul qildi

* no new providers can apply

---

# ✅ 1. Backend Validation Logic (CRITICAL)

Before creating an offer:

## Step 1 — Check request status

If request:

* matched
* completed
* cancelled
* closed

→ block new offer creation.

---

## Step 2 — Count active offers

```ts id="jlwm9t"
const activeOffers = offers.filter(
  o =>
    o.requestId === request.id &&
    ["pending", "negotiating", "accepted"].includes(o.status)
);
```

---

## Validation:

```ts id="h2uvlu"
if (activeOffers.length >= 5) {
  throw new Error("Active offer limit reached");
}
```

---

# ✅ 2. Lifetime Offer Protection

Count ALL offers ever created for request:

```ts id="0w2xyu"
const totalOffers = offers.filter(
  o => o.requestId === request.id
);
```

---

## Validation:

```ts id="mjlwm7"
if (totalOffers.length >= 10) {
  throw new Error("Lifetime offer limit reached");
}
```
