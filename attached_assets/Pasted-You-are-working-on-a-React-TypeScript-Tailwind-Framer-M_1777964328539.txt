# ✅ 1. Create Reusable Component

Create a new component:

```text
/components/ui/RollingCategories.tsx
```

---

## Component Requirements:

### Props:

```ts
type Props = {
  items: string[];
  interval?: number; // default 3000ms
};
```

---

## Behavior:

* Display ONE category at a time
* Every `interval` ms:

  * next category appears
* Loop infinitely
* When clicked one of the props navigate to survey with automatically choosing that selected category.
---

## Animation:

Use **Framer Motion**

### Effect:

* New item slides **from bottom → up**
* Old item fades/slides out

---

## Implementation (Required Behavior):

* Container:

  * fixed height (prevent layout shift)
  * `overflow-hidden`

* Animated item:

  * `position: absolute`
